BopBot

A fully-featured self-hosted music bot instance for firefly and friends based off JMusicBot

Adding to your server

BopBot is currently a private bot for friends of Adrian, so if you're interested in adding it, send a DM to aidswidjaja or email bopbot@adrian.id.au - thanks!

Bot status

Please visit status.adrian.id.au

Command list

Use the -help command to figure out what these actually do.

  // General commands
  settings = [ status ]
  changelog = []

  // Music commands
  bassboost = [ bb ]
  clear = [ stop ]
  lyrics = [ ly ]
  nowplaying = [ np, current ]
  play = [ p ]
  playlists = [ pls ]
  queue = [ q, list ]
  remove = [ rm, delete ]
  scsearch = [ soundcloud ]
  search = [ youtube, s ]
  shuffle = [ sh ]
  skip = [ voteskip, n ]

  // Admin commands
  bbadmin = []
  prefix = [ setprefix ]
  setdj = []
  settc = []
  setvc = []

  // DJ Commands
  forceremove = [ fr ]
  forceskip = [ fs ]
  move = [ mv, movetrack ]
  pause = [ ps ]
  playnext = [ pop ]
  repeat = [ r, l ]
  jump = [ skipto ]
  disconnect = [ dc, die ]
  volume = [ v, vol ]
